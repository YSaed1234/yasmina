**ياسمينة** هي منصة تجارة إلكترونية متكاملة تدعم تعدد البائعين (Multi-vendor)، مصممة لتقديم تجربة تسوق سلسة وفاخرة مع لوحة تحكم إدارية قوية وشاملة.

---

## ✨ المميزات الرئيسية (Core Features)

### 🛒 تجربة التسوق (Storefront)
- **دعم تعدد البائعين**: منصة تتيح لمختلف البائعين عرض منتجاتهم وإدارة مبيعاتهم بشكل مستقل.
- **إدارة السلة والمشتريات**: نظام سلة تسوق ذكي مع دعم الكوبونات والخصومات التلقائية.
- **نظام العروض والترقيات**: قسم خاص بالعروض الترويجية والخصومات الحصرية.
- **قائمة الأمنيات (Wishlist)**: إمكانية حفظ المنتجات المفضلة للرجوع إليها لاحقاً.
- **التقارير والمراجعات**: نظام تقييم للمنتجات يتيح للمستخدمين مشاركة تجاربهم.

### 👤 حساب المستخدم والمحفظة (User Profile & Wallet)
- **المحفظة الرقمية (Digital Wallet)**: نظام رصيد داخلي يتيح للمستخدمين الشحن والشراء المباشر.
- **نظام الولاء (Loyalty Points)**: كسب نقاط مع كل عملية شراء وإمكانية تحويلها إلى رصيد في المحفظة.
- **إدارة العناوين**: نظام متطور لإدارة عناوين الشحن المتعددة وربطها بالمناطق والمحافظات.
- **تتبع الطلبات**: واجهة لمتابعة حالة الطلبات من لحظة التأكيد وحتى التسليم.

### 🛠️ لوحة التحكم الإدارية (Admin Dashboard)
- **إدارة المنتجات والمخزون**: تحكم كامل في إضافة وتعديل المنتجات وتحديث كميات المخزون.
- **إدارة الطلبات والمرتجعات**: نظام لمعالجة الطلبات، تعيين السائقين، وإدارة طلبات الإرجاع (Return Requests).
- **التقارير والتحليلات**: تقارير مفصلة عن المبيعات، المخزون، حركة الزوار، وأداء البائعين.
- **إدارة المستخدمين والأدوار**: نظام صلاحيات متطور (Roles & Permissions) للتحكم في وصول الموظفين.
- **إدارة الشحن والخدمات اللوجستية**: تعريف المحافظات، المناطق، وتكاليف الشحن المتغيرة.

## 🛠️ التقنيات المستخدمة (Tech Stack)

- **Backend**: [Laravel 12](https://laravel.com) - إطار عمل PHP قوي وآمن.
- **Frontend**: [Blade Templates](https://laravel.com/docs/blade) مع [Tailwind CSS](https://tailwindcss.com) للتصميم المتجاوب.
- **Asset Bundling**: [Vite 7](https://vitejs.dev) لسرعة التطوير والبناء.
- **Database**: MySQL/MariaDB.
- **Architecture**: Modular Architecture (تقسيم المشروع إلى وحدات مستقلة: Admin, Web, Vendor).

## 🚀 التنصيب والتشغيل (Installation)

1. **تحميل المشروع**:
   ```bash
   git clone <repository-url>
   cd yasmina
   ```

2. **تثبيت الاعتمادات (Dependencies)**:
   ```bash
   composer install
   npm install
   ```

3. **إعداد ملف البيئة**:
   ```bash
   cp .env.example .env
   php artisan key:generate
   ```

4. **إعداد قاعدة البيانات**:
   ```bash
   php artisan migrate --seed
   ```

5. **تشغيل المشروع**:
   ```bash
   php artisan serve
   npm run dev
   ```
